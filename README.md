# JavaCFPR
Exercices pour le cours d'enrichissement de Java au Centre de Formation Professionnelle des Riverains en vue d'une spécialisation en développement d'applications dans le cadre de la technique en informatique du Collège Montmorency.

## Structure

- ``althotth`` : exercices journaliers
    - ``atXX`` : exercices du jour XX
- ``commun`` : dossier fourni par l'enseignant
- ``lib`` : dossier fourni par l'enseignant
- ``sandbox`` : dossier fourni par l'enseignant
